摄影素材管理助手 - 用户说明

支持系统：
Windows 10 / Windows 11，64 位系统。

使用方式：
1. 先完整解压整个文件夹，不要只拖出单个启动文件。
2. 双击“启动.bat”运行。
3. 启动窗口会显示本地访问地址和日志位置。
4. 程序启动成功后会自动打开浏览器。如果没有自动打开，请在浏览器里访问启动窗口显示的地址。
5. 停止程序可以关闭启动窗口，也可以双击“停止.bat”。

首次运行：
1. 程序会先启动基础界面。
2. 本地语音识别、转写、语义检索所需的 AI 依赖会在后台自动安装。
3. 依赖安装后会统一下载并加载语音模型、语义模型，完成一次自检。
4. 准备完成前，入库处理、搜索、智能搜索等功能会暂时禁用。
5. 准备进度会在网页右下角以任务窗口显示，可以展开或缩小；完成后该窗口会自动消失。
6. 首次准备成功后，后续启动不会再显示自检等待；模型会在后台静默预热或在使用时按需加载。

下载和保存位置：
1. Python 依赖安装在：软件目录\python\Lib\site-packages
2. 模型、缓存、数据库、日志、音频切片保存在：软件目录\.pmm_data
3. 程序默认使用国内 pip 源；本地语义模型优先通过 ModelScope 国内源下载。

如果安装失败：
1. 先检查网络，然后重新双击“启动.bat”，程序会继续尝试安装。
2. 如果页面提示路径过长，请先关闭程序，把整个软件文件夹移动到更短路径，例如 C:\PMM 或 E:\PMM，然后重新双击“启动.bat”。
3. 日志文件位置：软件目录\.pmm_data\logs\startup.log
4. 语音模型、语义模型下载或加载问题，可查看：软件目录\.pmm_data\logs\model.log
5. 如果语义模型缓存损坏，程序会自动清理损坏缓存并重试下载。

注意：
1. 程序运行时不要移动或删除软件目录。
2. 如需清空本机数据，可以在程序停止后删除 .pmm_data 文件夹。
3. 如需更换大模型配置，可以编辑 .env 文件。
